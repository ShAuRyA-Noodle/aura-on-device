# _trust/ — Public Trust Pack

Public-facing artefacts that earn external trust for the Aura
Phase 1 submission. Built for Samsung EnnovateX 2026 by Galaxy Brain.

This folder is the "evidence pack" that lives next to the deck. It is
the set of artefacts a Samsung R&D engineer, a One UI design lead, a
Samsung Ventures partner, or an Indian VC would scan in two minutes
to decide whether the team is serious.

## Voice rules

Every artefact in this folder follows the locks in
`/Users/shauryapunj/Desktop/Samsung Hack/deck_spec.md` §0:

- Strict English. Indian context surfaces through named real
  artefacts (UPI, IRCTC, Zomato, BMTC), never through Hinglish.
- Banned words: empower, leverage, seamless, revolutionary,
  game-changing, paradigm, unleash, harness, synergy, holistic,
  robust, cutting-edge, AI-powered, transformative.
- No emojis except ₹.
- No exclamation marks.
- No marketing voice.

Confident, observational, slightly self-aware. Specificity beats
abstraction. Numbers, named apps, named moments.
