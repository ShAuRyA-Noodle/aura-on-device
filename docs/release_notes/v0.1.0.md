## Architecture commitments locked in this release

- **Three layers**: Sense, Intelligence, Experience.
- **Four agents**: Comms (Gemma 2B + LoRA), Calendar (rules + Phi-3-mini), Finance (regex + Gemma 2B + LoRA), Wellness (XGBoost + Phi-3-mini).
- **Orchestrator**: Phi-3-mini-4k-instruct Q4, LangGraph state machine, seven states.
- **Silence Budget**: 3 tokens/day default, refundable on user-marked usefulness, visible in Settings, Phase 2 KPI. ADR-0003.
- **Reasoning Trace**: schema fixed in `trace.v1.json`, drawer UI in five sections, retention 30 days default. ADR-0004.
- **Memory graph**: SQLite + sqlite-vss + SQLCipher AES-256, hash-chained audit log, daily Merkle root. ADR-0007.
- **On-device only**: no cloud egress on the hot path; export is share-sheet only. ADR-0005.
- **Platform strategy**: Apple-only Phase 1+2 reference build; Galaxy ecosystem as production target; iPhone Phase 3 demo unless venue device available. ADR-0006.

## Known limitations

- iOS sandbox blocks SMS read; FinanceAgent UPI feature is Android-only. Manual Gmail receipt import on iOS during pilot. (`plan.md` §27 R13)
- Custom keyboard on iOS does not enable "Allow Full Access"; entropy reading is narrower than the Android path. (`technical_spec.md` §10.9)
- Tier C devices (4–6 GB RAM) run a reduced-feature build with Gemma 2B as orchestrator and rule + DistilBERT for agents. (ADR-0002)
- IRB-style pilot consent without formal IRB approval — defensible for student submission, not for clinical claims. (ADR-0010)
