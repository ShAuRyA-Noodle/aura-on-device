# Release v1.0.0 — Aura Public Launch

Date: 2026-05-07.
Cut for: public launch as a real product, in parallel with the Samsung
EnnovateX 2026 submission. v1.0.0 is the union of Phase 1 blueprint,
Phase 2 prototype scaffolding, and Phase 3 finals readiness.

This is the cut external evaluators, press, and pilot users see.

---

## Why a 1.0.0

v0.1.0 was a blueprint cut. The deck, the architecture, the threat
model, and the privacy promises were locked but no code shipped. v1.0.0
adds the working orchestrator, the agent reference implementations, the
memory graph, the Wizard-of-Oz pilot kit, the HuggingFace Space, the
GitHub Pages site, the press kit, and the launch posts. The product is
real now; the next number is a Phase 2 minor cut after pilot data lands.

  https://huggingface.co/spaces/Shaurya-Noodle/Caramel_Coin.

## Constraints, named up front

- **Apple-only Phase 1 + 2 reference build.** Galaxy ecosystem is the
  production target. Every Sense API has a 1:1 Android counterpart on
  slide 4a of the deck.
- **₹2,000 total budget.** Print plus buffer. No flagship Galaxy phone
  in the lab. No paid Apple Developer Program. No cloud GPU.
- **Two-person team.** Galaxy Brain. No advisors, no contractors.

These constraints are not apologies. They are the design space the
product was built inside.
